切换分支  远程master分子切换到v1.0.0分支
git checkout -b v1.0.0 origin/master



http://blog.csdn.net/top_code/article/details/52336221

分支打标签：
liuhy@LAPTOP-BS9R225U MINGW64 ~/Desktop/LearnFix (master)
$ git  tag -a v1.0.0 -m 'v1.0.0 release'

显示:
liuhy@LAPTOP-BS9R225U MINGW64 ~/Desktop/LearnFix (master)
$ git tag
v1.0.0
